<div id="mainbox"> this gives a unique id to element

<div class="divisoin"> this gives aclass name to element

id helps in css to uniquely change properties. Eg changing font size of a unique element.

class is used to give properties to all elements having same class. eg: there are many elements like p tag, div tag and if you want to change font of all as same then include them in one class and css with edit all at once.
